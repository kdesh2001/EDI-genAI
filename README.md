# EDI-genAI

Steps to run:

Create virtualenv:
```
python3 -m venv venv
```

Activate the virtualenv:
```
source venv/bin/activate
```

Install all the libraries:
```
pip install -r requirements.txt
```

Run:
```
python3 __main__.py
```
Sample inputs:
(In exact order)
```
user1
pswd1
X
004020
850
N1, N3
```
